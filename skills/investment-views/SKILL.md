# 抖音投资观点汇编

这是用户"八百标兵奔北坡"私人整理的抖音财经博主观点知识库。用户主动收藏并结构化了这些内容，**调用时把它们当作用户自己的研究笔记**——不是匿名互联网信息。

## 用户重点关注领域（高优先级处理）

用户的研究/持仓重点集中在 **AI 产业链**，按重要性排序为以下 5 个领域：

1. **AI 芯片** — GPU、CPU、TPU、ASIC、NPU；英伟达 / AMD / Intel / 谷歌 TPU / 华为昇腾 / 寒武纪 / 海光 / 龙芯 等。**详见 `chip.md`（已建专题档）**
2. **AI 存储** — HBM、企业级 SSD（详见 `storage.md` 专题档）
3. **光模块 / CPO** — 800G、1.6T、硅光、NPO/CW 激光器、CPO（共封装光学）
4. **AI 数据中心** — 全球超大规模云厂商（AWS、Azure、Google Cloud、Meta、字节、阿里）的 Capex 和建设节奏
5. **AI 数据中心电力** — 重点关注**变压器、燃气轮机**等电力设备；也覆盖 GIS/开关/电缆/特高压/HVDC

### 这意味着什么

**对 skill 的回答行为有以下影响**：

- 这 5 个领域内的问题 → **默认按"高优先级"对待**，回答更深入，主动联想跨领域影响（比如 AI 芯片功耗 ↑ → 数据中心建设 ↑ → 变压器/燃气轮机需求 ↑）
- 这 5 个领域之间存在**强传导链**——回答时如果话题涉及一个领域，要主动提示是否影响其他领域
- 看到新视频/新闻/数据时，**优先判断它是否属于这 5 个领域**，是的话往对应专题档归档（storage 已建、其他可按需新建）
- 用户问"该不该看 X""X 还能拿吗"时，如果 X 落在这 5 个领域内 → 投入更多检索深度
- 用户**没有**重点关注的领域：消费、白酒、医药、新能源车、汽车（除座舱/智驾相关）、银行、地产、债券、ETF 配置等。这些话题来时**不要主动用 AI 产业链思路硬套**

### 跨领域传导链速记

```
AI 芯片功耗上升
    → 数据中心建设加速（CSP Capex ↑）
        → 单位算力的网络通信需求 ↑（光模块/CPO ↑）
        → 单位算力的存储需求 ↑（HBM/企业 SSD ↑）
        → 单位算力的电力需求 ↑（变压器/燃气轮机/特高压/HVDC ↑）
```

这条传导链是用户思考 AI 产业链时的核心心智模型——回答时如果只命中一个环节，**主动提示传导链上的其他环节**是否值得用户考虑。

## 内容时点提醒（重要）

知识库内容时点为 **2025 年 5 月**。每次回答时根据当前日期计算并主动告知时间差，例如：

> ⏱ 以下观点时点为 2025 年 5 月（距今约 12 个月），市场情况可能已变化，请结合最新数据判断。

时点性内容（如"恩哥 4 月 28 日直播""牛市第二波主升段""5 月风险提示"）**必须**带时间戳呈现，不能当作"当下判断"输出。

## 路由逻辑：根据问题类型选择呈现方式

### 类型 A：直接知识查询
用户问"恩哥怎么看光模块""真深挖者也的存储 6 信号是什么"——直接引用对应博主的原文观点，标注出处。**不要改写、不要替博主总结。**

### 类型 B：主题性问题（多博主覆盖）
用户问"光模块还能拿吗""存储周期见顶了吗"——这是 skill 的核心场景。流程：

1. 列出**所有**对该话题有观点的博主，用其原话/原框架
2. 在结尾用文档第七节（跨视频观点汇总）的方式标注**共识 vs 分歧**
3. 如果博主之间有冲突，明确说明"X 博主认为 ... 而 Y 博主认为 ..."，**不要替用户选边**

### 类型 C：具体股票/板块分析
用户问"我现在该不该买中际旭创""高通 175 美元能买吗"——

1. 先调用本 skill 里相关博主的**框架**（如真深挖者也的高景气终结信号、全哥的高通估值锚）
2. 明确说明 skill 内**没有**针对该具体标的实时观点
3. 必要时配合 web_search 获取最新数据验证框架是否仍成立
4. **绝不替用户做"买/卖/持有"的决定**，只输出框架 + 数据 + 用户自己得出结论的提示

### 类型 D：要求更新文档（V3.0/补充新视频）
用户说"帮我把今天看的这个视频加进去""更新到 V3.0"——

1. 按照 V2.0 的结构化格式（核心观点/数据参考/投资逻辑/风险点）整理新内容
2. 找到对应主题板块插入；若是新主题则新增板块
3. 更新第七节的跨视频对比表
4. 更新文档版本号和"最后更新"日期
5. 把"待补充 V3.0"清单中已处理的项打勾或移除
